# Examples

本目录包含 transcript-to-doc skill 的输入→输出示例，帮助新用户快速理解 skill 的能力和输出质量。

## 文件说明

| 文件 | 内容 |
|---|---|
| `input-transcript.md` | 一段简短的模拟录音稿（~5 分钟课程片段） |
| `output-document.md` | 对应的教学文档输出（按 template.md 框架生成） |

## 对比要点

对比输入和输出时，关注以下转化：

1. **口语 → 书面语**：口头禅消除、术语补全、句子结构化
2. **提纲 → 详案**：录音稿一句话 → 文档展开为一个完整小节
3. **踩坑保留**：老师随口说的 bug → 完整的排错实录段落
4. **结构注入**：原始录音没有目录/目标/练习 → 文档自动生成
5. **图示还原**：口述的流程 → ASCII 流程图
6. **类型匹配**：代码实操段用 Step-by-Step 模板，概念段用三层递进模板

## 使用方式

1. 阅读 `input-transcript.md`，想象你正在听这堂课
2. 阅读 `output-document.md`，看 skill 如何将口头内容转化为可用的课件
3. 对照 template.md 的规范，理解每个段落用了哪个 Pattern
